# Reiknivélar – BYKO Leiga

A collection of Icelandic construction calculators published via GitHub Pages.

## Live site

**<https://fannar1994.github.io/Reiknivelar/>**

The landing page (`index.html`) links to all available calculators:

| File | Calculator |
|------|------------|
| `vinnupalla-reiknivel.html` | Vinnupalla reiknivél |
| `motareiknivel-byko-v11.html` | Steypumótareiknivél |
| `loftastodir.html` | Loftastoðir |
| `Girdingar_reiknivel.html` | Iðnaðargirðinga reiknivél |
| `hjolapallar_reiknivel_sp.html` | Hjólapalla reiknivél |

## Deployment

The site is deployed automatically to GitHub Pages via the
`.github/workflows/static.yml` GitHub Actions workflow whenever changes are
pushed to `main`. No manual steps are required after merging a pull request.
